Expense Tracker
===============

A simple Django / Bootstrap / AngularJS based expense tracker.
Deployable as WSGI application or on Google App Engine / Cloud SQL.

### Front end 
* Bootstrap 3.0
* jQuery 2.0
* AngularJS 1.2
